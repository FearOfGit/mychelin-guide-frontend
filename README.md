## Commit Convetions

| Message  | Description                   |
| -------- | ----------------------------- |
| fix      | fixed bugs and errors         |
| feat     | added new feature             |
| refactor | updated existing feature/code |
| style    | fixed styles                  |
| docs     | added/updated docs            |
| add      | added icons, images, etc.     |
| ver      | updated version               |
